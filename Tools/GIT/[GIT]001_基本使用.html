简单介绍

git 和 github什么关系？

  • git: git 是一个强大的管理工具，它是工具

  • github: github 是平台，提供一个完整远程仓库，并且可以和其他gist进行交流，相互学习。

远程推送

即把 本地仓库的 project的东西提交到 github的myProject。

注意：本地与github是通过 ssh 进行加密会话的，所以需要一个共同的key

一、先配置

  1. 本地生成 key

    打开 c/Users/Administrator/.ssh/id_rsa 文件，复制。

  2. github设置key

    在 Settings--> SSH and GPG keys --> New SSH Key,粘贴 key

  3. 测试
