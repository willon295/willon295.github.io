模糊查询

LIKE ： 模糊查询

  • _: 一个字符

  • %: 任意字符

  • \: 转义使用，\_ 就是 _, 一般配合 escape 使用

UNION,UNION ALL

  • UNION: 将结果进行合并，去重
  • UNION ALL: 直接将结果进行合并，不去重

minus , intersect

minus : 求差集，第一个减去第二个
intersect : 求交集

group by

分组: select 查询出现的字段一定要在group by中出现，否则编译不通过

如果要对组函数进行约束，必须使用 having, 不能 使用 where 进行约束
