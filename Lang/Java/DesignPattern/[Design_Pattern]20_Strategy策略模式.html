[Design_Pattern]20_Strategy策略模式

2018-06-10
  • 设计模式

策略模式 是 行为模式 的一种，当一个类的行为或者是算法在运行时可以改变。

举个简单的例子， 使用第三方接口解析数据，但是第三方有JSON、有XML，如果使用if..else就很low

如果此时需要其他类型的数据，那么扩展则需要修改 if..else结构

实现流程

  1. 将具体策略与环境分离
  2. 建立抽象的策略接口/基类
  3. 具体策略实现
  4. 让环境持有一个策略的引用
  5. 在不同的环境下调用不同的策略并且执行
