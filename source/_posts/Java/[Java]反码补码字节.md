# 源码反码补码

1. 源码
二进制表示
2. 反码
源码的符号位不变其他位 `加1`
3. 补码
源码 `取反加1` ， 内存中的数据用补码保存

# 返回值

1. `基本运算符`  ，`三目运算符`  的 运算结果 必须要有 `变量接收` 或者 `将其返回` ，否则编译不通过。

```
3+4;      //报错
3>4?1:0;  //报错
```
2. `调用函数` ，若有返回值 `可以不用变量接收`
```
add(3,4); 
int  a = add(3,4);
```
