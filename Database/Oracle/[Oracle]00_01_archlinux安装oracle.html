Oracle 软件安装

  1. 从AUR安装

  2. 修改 pkgbuild 文件

  • 删除下载文件 oracle-xe-11.2.0-1.0.x86_64.rpm 相关的行
  • 删除解压 oracle-xe-11.2.0-1.0.x86_64.rpm 相关的行
  • 删除 md5 矩阵内的 第一行
  1. 从 这里下载 oracle-xe-11.2.0-1.0.x86_64.rpm 文件
  2. 解压该rpm包内的文件到 /tmp/yao-tmp-用户名/aur-orcle-xe-11g/src

  3. 修改 /tmp/yao-tmp-用户名/aur-orcle-xe-11g/src 目录下所有文件的owner

  4. 继续执行， 在构建打包完成之后， oracle安装完成， 进入后续配置

Oracle 后续配置

  1. 将 root 、当前用户 加入 dba 组

  2. oracle初始化配置, 输入命令之后默认回车即可

  3. 配置 ORACLE_HOME , ORACLE_SID 环境变量

Oracle 服务管理

  1. 开始、重启等操作
  1. 登陆
