全注解方式

Mapper的自动装配有几种方式：

  1. 每个 Mapper上使用 @Mapper 注解
  2. 不用 @Mappr 注解 ， 在入口使用 @MapperScan(basepackages="XXX") 指定Mapper位置

XML 文件方式

  2. 写 XXmapper.xml 文件

  3. 配置文件，取别名，指定xml文件位置

  4. 在 入口处 使用 @MapperScan 指定 接口 地址
