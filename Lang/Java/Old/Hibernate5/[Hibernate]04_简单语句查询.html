  • 面向对象
  • 关键字比如 SELECT ，FROM 和 WHERE 等不区分大小写

常用的方法

  1. avg(propertyName) : 属性的平均值

  3. max(propertyName) : 属性值的最大值
  4. min(propertyName ) : 属性值的最小值
  5. sum(propertyName) : 属性值的总和
