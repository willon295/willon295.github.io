PhpStorm的坑

  1. 默认服务启动，端口为65442
  2. 无法获取post提交的数据，get可以.

解决办法：

  1. 使用wamp 作为运行环境，phpstorm 只用作开发环境
  2. 抛弃 phpstorm,直接用sublime敲代码（效率明显低一点

phpstorm设置

  1. 找到Settings->Deployment
  2. 点 +号，填入名称，选择In place.
  3. 点击刚配置的名称Web Server root URL 填入host解析的域名
