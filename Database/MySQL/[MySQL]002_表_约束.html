完整性约束

  • NOT NULL: 字段的值不为空
  • DEFAULT: 字段的默认值
  • UNIQUE KEY (UK): 值的唯一性
  • PRIMARY KEY (PK): 表的主键
  • AUTO_INCREMENT: 字段值自增
  • FOREIGN KEY (FK): 外键

举个栗子
