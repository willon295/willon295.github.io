Maven项目依赖

主要依赖

  1. junit： 单元测试
  2. Struts2-core: struts核心
  3. hibernate-core: hibernate核心
  4. spring-core: spring核心
  5. spring-struts: spring-struts整合插件
  6. spring-hibernate: spring-hibernate整合插件
  7. spring-tx: spring事务管理
  8. spring-orm: orm
  9. aspectj: aop依赖
  10. c3p0：数据库连接池
  11. mysql-connector-java: 数据库驱动
