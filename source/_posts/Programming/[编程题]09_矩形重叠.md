# Problems

输入矩阵的对角线两个点的坐标，如： `{x1，y1,x2,y2}` ，判断两个矩形是否重叠

# Example


```
int a[] = {0,0,1,1};
int b[] = {2,2,4,4};
//false

int a[] = {0,0,4,4};
int b[] = {2,2,5,6};
//true
```


# Coding


    

```
public class Main {
    /**
     * 判断两个矩形是否重叠
     * @param a 矩形a
     * @param b 矩形b
     * @return 是否重叠
     */
    public static boolean isRecOverload(int a[], int[] b) {
        // a左下角  ,  b 右上角比 ， a 在 b 的 左下
        // a右上角 ，  b 左下角比 ， a 在 b 的 右上
        return a[0] < b[2] && a[1] < b[3] && a[2] > b[0] && a[3] > b[1];
    }
}


```
