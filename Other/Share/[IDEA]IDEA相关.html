科学激活

  1. 将JetBrainsCracker-2.xx.jar 复制到idea安装目录bin下

  2. 编辑文件 bin/idea.vmoptions 和bin/idea64.vmoptions ，最后一行加入

  3. 启动时写入激活key

  4. 结束

添加自定义快捷输入

  1. File -> Settings -> Live Templates -> + Live Template
  2. Abbreviation 填写 提示语, 如: loge

  3. Template text 填写表达式

  4. variables 变量属性填写 groovy脚本
