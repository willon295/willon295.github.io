# Hash ?



哈希用于将 `任意长度`  的数据映射到 `固定长度` 的数据

1. 哈希函数:  用于实现hash的算法的函数
2. 哈希值: 哈希函数的返回值



# Hash 冲突

在使用哈希函数时,  `不同的输入` 得到了 `相同的输出`  , 即出现了相同的 hash 值



- **链地址法**： 桶/数组的元素节点使用链表之类的实现, 让冲突的元素直接存入链表地址的最后一个.
- **开放地址法**： 
1. 如果ｈａｓｈ存在:　在通过其他的算法　，　算出新的地址
2. 判断新的地址是否可用
3. 可用，　存入，不可用，　再计算
4. 直到计算出新的可用地址

- 建立公共溢出区

  建立公共溢出区:  就是建立一个公共表, 把冲突的数据放入溢出表中
